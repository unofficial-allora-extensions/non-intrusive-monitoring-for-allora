Note: the inclusion of the "WiFi_interface" is only for the sake of completeness, as it was used in early phases of the project and constitutes an important resource of the original AlLoRa repository (https://github.com/SMARTLAGOON/SMLG_AlLoRa). In the final version, the communication between the AlLoRa Gateway and the AlLoRa Adapter is a UART Serial one, so it instead uses the "Serial_interface", and the WiFi one is not needed.
