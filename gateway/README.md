The code provided in this folder is meant to be used on a Raspberry Pi that serves as a AlLoRa Gateway, with its access to the AlLoRa channel provided by a AlLoRa Adapter run by a LilyGO T3S3 -with a serial UART connection between the two-.

Some classes from a previous development state of the project have been preserved, which served the purpose of offering a simple HTML "Dashboard" before the construction of the Grafana solution was made. As it can allow for a specific monitorization and supervision of the terminal nodes assigned to a determined gateway (and not a centralized approach like the definitive one developed, in the sense of covering the administration of all the terminal nodes in one place), it has been considered that its preservation is still of interest, at least for troubleshooting and testing.
