Note: it's important to state the "short\_mac" field as "false" in **ALL "LoRa.json" files** used for AlLoRa communication. Mixed configurations (i.e. nodes with "short\_mac" as "true" and others as "false") have been proved to be **incompatible**, especially in the interactions where some kind of AlLoRa Gateway -from the oficial repository (https://github.com/SMARTLAGOON/SMLG\_AlLoRa)- is used.
